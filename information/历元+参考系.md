## 详细核实说明

### 1. AC (I/154) - Astrographic Catalogue
**ReadMe来源**: `ac/ReadMe`

- **参考系**: B1950
- **历元**: **~1907年**（加权平均；各板历元约1892-1950不等）
- **核实**: ReadMe明确说明 "Epoch plate epoch" 字段记录各星的板历元
- **重要说明**: AC坐标保留原始观测历元，未进行标准化归算到B1950.0

### 2. AGK1 (I/310) - 原文有误，已修正
**ReadMe来源**: `agk1/ReadMe`

- **参考系**: **1875**（非FK4）
- **历元**: **1876-1899**（观测年份）
- **核实**: ReadMe明确说明 "Right ascension 1875"，坐标系是1875年参考系
- **修正**: 原文"未明确"是错误的

### 3. AGK3 (I/61B)
**ReadMe来源**: `agk3/ReadMe`

- **参考系**: FK4
- **历元**: B1950.0
- **核实**: ReadMe明确 "Right ascension (1950.0)"，位置归算到FK4系统B1950.0
- **备注**: 观测于1956-1964年，但位置已归算到B1950.0

### 4. FK4 (I/143)
**ReadMe来源**: `fk4catalogue/ReadMe`

- **参考系**: FK4
- **历元**: B1950.0（主文件），另有B1955/1960/1965/1970/1975版本
- **核实**: ReadMe明确说明 "RA B1950"，FK4定义了B1950参考系

### 5. SAO (I/131A)
**ReadMe来源**: `sao/ReadMe`

- **参考系**: B1950 和 J2000（双系统）
- **历元**: 1950.0 和 J2000.0
- **核实**: ReadMe说明同时提供B1950.0 FK4和J2000.0 FK5坐标

### 6. ACRS (I/171)
**ReadMe来源**: `acrs/ReadMe`

- **参考系**: FK4 → FK5
- **历元**: B1950.0 → J2000.0
- **核实**: ReadMe说明 "compiled on B1950.0 FK4 and then transformed to J2000.0 FK5"

### 7. ACT (I/246)
**ReadMe来源**: `act/ReadMe`

- **参考系**: ICRS（Hipparcos System）
- **历元**: J2000.0
- **核实**: ReadMe明确 "on the Hipparcos System (J2000.0) for epoch J2000.0"

### 8. PPM (I/146+I/193)
**ReadMe来源**: `ppm/North/ReadMe`

- **参考系**: FK5
- **历元**: J2000.0
- **核实**: ReadMe说明 "Right Ascension for the Equinox=J2000.0 and Epoch=J2000.0, on the system of FK5"

### 9. Yale (I/141) - 原文有误，已修正
**ReadMe来源**: `yale/ReadMe`

- **参考系**: B1950
- **历元**: **观测历元**（各星不同，非标准化）
- **核实**: ReadMe说明 "The coordinates on the B1950 system are given for the epoch, Ep. Proper motions must be applied to change the epoch."
- **修正**: 原文说"B1950.0 / J2000.0"历元是错误的，Yale星表的位置是在各星的观测历元

## 参考系说明

| 参考系 | 说明 |
|--------|------|
| B1950 / FK4 | 旧的基本参考系，基于FK4星表 |
| FK5 | 第五基本星表系统，比FK4更精确 |
| ICRS | 国际天球参考系，现代标准，与J2000.0一致 |
| J2000 | 常指ICRS，有时指FK5 J2000 |
| 1875 | 历史参考系，AGK1使用 |

## 历元说明

| 类型 | 说明 |
|------|------|
| 标准历元 | 位置已归算到标准历元（如B1950.0, J2000.0），使用自行可推算到任意历元 |
| 观测历元 | 位置在原始观测时刻，各星历元可能不同 |
| 板历元 | 照相星表的观测历元，取决于拍摄时间 |
