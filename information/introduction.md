# 引言

天然卫星的精密轨道确定是行星系统动力学研究和长期星历预报的重要基础，其中光学天体测量观测提供了关键的位置约束。
长期以来，地基观测积累了大量天然卫星位置数据，为轨道改进与相关研究提供了基础。
本团队自1986年至2024年持续开展天然卫星观测工作，累计获得约51170条位置测量数据，观测对象涵盖木星、土星、天王星及海王星系统。
上述观测数据已纳入 IMCCE 维护的天然卫星数据体系（NSDC），并与历史观测共同构成统一的数据基础。
在相关轨道改进研究中，团队已开展包括土星与海王星天然卫星在内的多目标轨道分析工作，并依赖 NSDC 中汇集的全球观测数据。
在实际处理过程中发现，不同历史观测所采用的参考星表存在系统性差异，成为限制数据统一利用的重要因素。

在光学天体测量数据处理中，参考星表引入的系统误差（catalog bias）是影响观测一致性的重要因素。
由于历史观测通常依赖于当时可用的参考星表，不同星表在参考框架、系统误差及自行精度等方面存在差异，从而在多源观测数据中引入空间相关的系统性偏差。
这一问题在长期积累的观测数据中尤为突出。例如，NSDC 汇集了自19世纪以来的大量地基观测记录，其数据来源复杂，进一步放大了参考星表不一致所带来的系统误差问题。

针对 catalog bias 的影响，小天体天体测量领域已发展出较为系统的修正方法。
Chesley 等首先定量揭示了区域性星表偏差对轨道拟合的影响，
随后 Farnocchia 等在此基础上引入自行修正以改进相关处理方法。
随着 Gaia DR2 的发布，Eggl 等基于统一参考框架重新估计了多种历史星表的位置与自行系统误差，并构建了相应的修正表，为历史观测数据的统一处理提供了标准化手段。上述方法主要应用于小行星等小天体观测数据。

在天然卫星领域，相关方法已开始得到应用。
袁烨等基于 Gaia DR2 参考框架，对海王星天然卫星的地基观测进行了统一整理，并在数据处理中采用了 Eggl 等提供的星表修正结果，同时按照类似方法对部分未覆盖星表（如 YZC 和 Tycho-1）进行了补充修正，从而扩展了相关修正表在天然卫星观测中的适用范围。

在此基础上，本文在 Gaia DR3 参考框架下，对历史观测星表系统偏差进行了重新估计。
本文总体沿用既有去偏方法框架，但针对天然卫星观测数据的特点，
对交叉匹配过程中的星等筛选、匹配判据及冲突处理策略进行了量化约束，并对空间平滑过程中的基函数选择与参数设置进行了明确化，从而提高修正结果的一致性与可重复性。

<!-- 本文工作主要是在既有方法框架下的实现细化与参数明确化，而非构建新的去偏模型体系。 -->

本文构建的修正模型通过观测位置与 DE441 历表之间的残差（O−C）进行评估。通过比较修正前后的残差分布，可以检验修正表在降低系统性偏差方面的有效性。
